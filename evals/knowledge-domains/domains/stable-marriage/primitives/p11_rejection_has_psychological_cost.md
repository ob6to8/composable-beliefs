# p11: Rejection carries real psychological costs the algorithm doesn't model

Being rejected is painful. Repeated rejection can erode confidence, willingness to ask again, and self-perception. The algorithm treats each proposal as equally cheap. Humans have finite psychological reserves for rejection.

This is essentially uncontestable as an observation. The question is whether the cost is large enough to change the optimal strategy.
